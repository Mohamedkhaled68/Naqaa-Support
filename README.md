# Naqaa Support Page

This is a simple support page for the mobile app **Naqaa**, built with Next.js and Tailwind CSS.

## Features

-   Responsive, centered layout
-   App name at the top: "Naqaa Support"
-   Contact section with email and phone
-   FAQ section with 3 collapsible questions and answers
